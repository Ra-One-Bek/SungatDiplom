# ⚽ Atletico Madrid Football Analytics App

Дипломная работа:  
Разработка веб-приложения для анализа спортивных показателей и статистики футболистов.

## 🛠 Технологии

### Frontend
- React + TypeScript
- TailwindCSS
- Vite

### Backend
- NestJS
- REST API
- API-Football (external data)

## 📊 Возможности

- 📋 Просмотр игроков и статистики
- 📈 Анализ формы игроков (custom model)
- 🧠 AI-рекомендации по составу
- 🏟 Управление составом (как в FIFA)
- 🔄 Реальные данные (API-Football)
- 🏥 Учет травм игроков
- ⚙️ Выбор схемы (4-3-3 и др.)

## 🤖 AI функции

- Анализ состава
- Определение слабых позиций
- Рекомендации по заменам
- Советы по тренировкам
- Проверка соответствия позиции
